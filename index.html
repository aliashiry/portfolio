أبني تطبيقات موبايل ومنصات ويب بـ Flutter و .NET

علي حسن علي — مطور Flutter وFull Stack .NET بخبرة عملية في بناء حلول متكاملة من الواجهة إلى قاعدة البيانات. أطبّق Clean Architecture وBLoC/Cubit في الموبايل، وASP.NET Core وAngular في الويب، مع CI/CD عبر Azure DevOps.

استعرض المشاريع سيرة Flutter ↓ سيرة .NET ↓

نبذة عني

مطور Flutter بخبرة سنتين في بناء تطبيقات iOS وAndroid باستخدام BLoC/Cubit وClean Architecture، مع تكامل RESTful APIs عبر Dio وFirebase للإشعارات والمصادقة. أربعة تطبيقات منشورة على Google Play في مجالات التعليم والتجارة والتوريد.

بالتوازي، أطوّر أنظمة ويب وخلفيات بـ ASP.NET Core Web API وAngular وEntity Framework Core، شاملة مشروع CRM عقاري ثنائي اللغة ومشروع تخرج بنظام MEP مدعوم بالذكاء الاصطناعي. أعمل حالياً في Forgotten Mines على تطبيقات Flutter Web وخدمات .NET مع خطوط CI/CD آلية.
